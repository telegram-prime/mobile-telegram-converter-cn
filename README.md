# MTCR Converter - Mobile Telegram 到 SESSION 或 TDATA 的转换器
## 设计用于将 Mobile Telegram 帐户转换为 SESSION+JSON（Telethon 或 Pyrogram）格式或 TDATA 格式。

 * Description also available in English. See: [MTCR Coverter ENG](https://github.com/telegram-prime/mobile-telegram-converter)
 * Описание программы так же доступно на русском языке. См. [MTCR Coverter RUS](https://github.com/telegram-prime/mobile-telegram-converter-RU/)

## 功能：
 - 将 Mobile Telegram 转换为 SESSION+JSON 或 TDATA 格式。
 - 输入支持两种格式：官方 Telegram (tgnet.dat) 和 Telegram-X (td.binlog)。
 - 多线程工作允许设置多个并发运行的线程。
 - 允许选择是否将帐户保留在当前会话中或登录到新会话。
 - 支持在转换或重新授权期间禁用帐户上的其他会话选项。
 - 选项可以设置自定义的连接重试次数和连接延迟。
 - 通过 2FA 输入 TXT 文件自动导入帐户的 2FA 或手动输入单个 2FA 用于所有转换后的帐户。
 - 灵活配置用于连接到会话的设备的模拟（旧会话和新会话不同）。
 - 为每个会话连接使用唯一的 HTTP 或 SOCKS5 代理 (支持多行)。
 - 将有效的、无效的、超时以及 2FA 不良帐户分别排序到目标目录的不同文件夹中。
 - 在操作之前备份源文件。
 - 提供全面的日志记录到“程序控制台日志”窗口，允许用户监控进度并排除任何问题。
