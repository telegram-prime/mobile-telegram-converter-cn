# MTCR Converter  
**移动版 Telegram → SESSION+JSON 或 TDATA**

> 现代、安全、高速的 Telegram 移动端账号转换工具

将 **官方 Telegram (tgnet.dat)** 和 **Telegram X (td.binlog)** 文件转换为：

- **SESSION+JSON**（兼容 Telethon / Pyrogram）  
- **TDATA**（Telegram Desktop / Telegram Prime 格式）

### 🌐 支持多语言
- English – [English Page](https://github.com/telegram-prime/mobile-telegram-converter/)  
- Русский – [MTCR Converter RU](https://github.com/telegram-prime/mobile-telegram-converter-RU/)  
- 简体中文 – 本页面

### ⚙️ 核心功能

| 功能                                 | 描述                                                                        |
|--------------------------------------|-----------------------------------------------------------------------------|
| 输入格式                             | 官方 Telegram (tgnet.dat) + Telegram X (td.binlog)                          |
| 输出格式                             | SESSION+JSON (Telethon/Pyrogram) 或 TDATA                                   |
| 多线程支持                           | 可设置任意数量的并发转换线程                                                |
| 会话处理方式                         | 保留当前会话 或 执行全新登录                                                |
| 会话清理                             | 可选择在转换期间终止账号所有其他活跃会话                                    |
| 连接参数调整                         | 自定义重试次数 + 每次尝试之间的延迟                                         |
| 两步验证处理                         | 从 TXT 文件自动导入 或 为所有账号手动输入同一验证码                         |
| 设备模拟                             | 灵活的 Android/iOS/桌面 设备模拟（新旧会话可分别设置）                     |
| 代理支持                             | 每个线程使用独立 HTTP/SOCKS5 代理（支持多行列表）                           |
| 结果分类                             | Valid / Invalid / Timed-out / Bad 2FA → 分别存入不同文件夹                  |
| 源文件备份                           | 操作前自动备份原始文件                                                      |
| 详细日志                             | 实时进度与错误显示在「Program Console Log」窗口                             |
